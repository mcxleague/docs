## Tech Stack


MCXLEAGUE's codebase is fully open-source and is built off of https://dub.co and is mode on top of the following technologies:

#### Next.js - Framework
#### Typescript - Language
#### Tailwind - CSS
#### Upstash - Redis
#### Tinybird - Analytics provider
#### PlanetScale - Database
#### NextAuth.js - Authentication
#### Turborepo - Monorepo
#### Stripe - Payments
#### Recend - Emails
#### Vercel - Deoployments
